The idea in one line

A specialist for every marketing job.

Skills are markdown files that hand an AI agent the right framework, checklist, and best practices the moment you start a marketing task. Instead of one blank prompt, you get the conversion playbook for a pricing page, the deliverability rules for a cold-email sequence, or the schema spec for an SEO fix — applied automatically.

◆

One shared foundation

product-marketing captures your positioning and audience once. Every other skill reads it first, so advice fits your product — not a generic template.

⚙

Triggered, not searched

Each skill lists the phrases that summon it. Say "this landing page isn't converting" and the CRO skill loads itself — no command to remember.

∞

They build on each other

copywriting ↔ cro ↔ ab-testing, seo-audit ↔ schema ↔ ai-seo. Skills cross-reference so a task pulls in everything it needs.

Two minutes

Install

Drop the whole library into any agent, or cherry-pick the skills you want. Files land in .agents/skills/ and symlink into .claude/skills/ for Claude Code.

terminal — npx skills
# install everything
npx skills add coreyhaines31/marketingskills

# or cherry-pick
npx skills add coreyhaines31/marketingskills --skill cro copywriting ads

# list what's available
npx skills add coreyhaines31/marketingskills --list

# Claude Code plugin install (alternative)
/plugin marketplace add coreyhaines31/marketingskills → /plugin install marketing-skills
